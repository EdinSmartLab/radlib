gas weight (sum to one)

THIS IS THE CLASS INTERFACE FUNCTION Given the gas state, set the k and a vectors.

These can then be accessed by the user. return through arg list the local gray gas coefficients (kabs) and the local weights (awts).

Parameters
Tinput: gas temperature
P_not_usedinput: Pressure (Pa)
xH2Oinput: mole fraction H2O
xCO2input: mole fraction CO2
xCOinput: mole fraction CO
xCH4_not_usedinput: mole fraction CH4
fvsootinput: soot volume fraction = rho*Ysoot/rhosoot
kabsoutput: absorption coefficients (1/m); size = 1 here (1 gas)
awtsoutput: weights (unitless; sums to 1); size = 1 here (1 gas)
