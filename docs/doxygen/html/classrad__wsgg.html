THIS IS THE CLASS INTERFACE FUNCTION Given the gas state, set the k and a vectors.

gas weight (sum to one)

These can then be accessed by the user. return through arg list the local gray gas coefficients (kabs) and the local weights (awts).

Parameters
Tinput: gas temperature (K)
Pinput: pressure (Pa)
xH2Oinput: mole fraction H2O
xCO2input: mole fraction CO2
xCOinput: mole fraction CO HERE FOR THE INTERFACE, NOT USED (... pass in 0.0)
xCH4input: mole fraction CH4 HERE FOR THE INTERFACE, NOT USED (... pass in 0.0)
fvsootinput: soot volume fraction = rho*Ysoot/rhosoot
kabsoutput: absorption coefficients (1/m) for nGG+1 (nGG gray gases + clear gas)
awtsoutput: weights (unitless; sums to 1) for nGG+1 (nGG gray gases + clear gas)
